Git

Git adalah sebuah tool yang sangat membantu developer, khususnya developer yang tidak bekerja sendiri dan perlu untuk melakukan kolaborasi. Git adalah salah satu contoh sistem Version Control System terdistribusi yang bersifat gratis dan open-source. Dimana Version Control System sendiri adalah tools yang membantu developer untuk mengelola perubahan source code dari waktu ke waktu, karena tentu saja developer akan selalu melakukan perubahan source code. Git akan "memantau" semua perubahan yang terjadi pada file dan Sehingga Git akan sangat mempermudah kolaborasi tim.

GitHub

GitHub adalah platform yang digunakan sebagai wadah developer untuk mengembangkan perangkat lunak. GitHub menggunakan Git sebagai Version Control System-nya. GitHub bisa diibaratkan sebagai sosial media bagi para developer. Digunakan untuk menyimpan berbagai source code proyek yang dimiliki oleh developer. Selayaknya sosial media, tentu GitHub memfasilitasi penggunanya untuk berkomunikasi dengan orang yang memiliki satu visi misi. Banyak proyek open source menggunakan Github untuk menyimpan Git, memecahkan masalah, menyusun kode, dan hal-hal lainnya.
